Parking of private car in Helsinki Capital Region: compare Helsinki Region Travel Time Matrix 2018 and thesis survey results of Sampo Vesanen

Focusing on private cars, this application visualises the parking time differences between Helsinki Region Travel Time Matrix 2018 (Tenkanen et al. 2018) and the survey data carried out for the master's thesis of Sampo Vesanen. All input data is aggregated to the level of postal code areas.

In this application terms origin postal code area and destination postal code area are used. Origin is the postal code you enter to the toolbar field on the left and is shown as Origin postal code area item in the map legend. Destination is the postal code area your cursor currently hovers on.

Please note that when using this application on desktop computer, you can activate map view zoom with the icon that appears at top right when hovering cursor over the map. Use double-click or mouse wheel to zoom. Click icon to return to the default view.

Tooltip column names

Column name Description
rush hour Rush hour traffic (09:00–11.00, 15.00–17.00)
midday Midday traffic (09.00–15.00)
all, average An average of all available data. For TTM18, these are Rush hour traffic, Midday traffic, and Route following speed limits without any additional impedances. For thesis survey data, these are the values of the survey question timeofday: Weekday, rush hour, Weekday, other than rush hour, Weekend, and Can't specify, no usual time.

Travel Time Matrix 2018, row names

Row name Description
sfp Time consumed in searching for parking. In TTM18, this is 0.42 minutes for all YKR IDs (in the case of this application, postal code areas) in the entirety of Helsinki Capital Region (Toivonen et al. 2014).
wtd_avg An averaged value (per postal code area, unit minutes) of walking time from one's parked private car to the final destination of the travel chain. In TTM18 this is 2.0 minutes for all YKR IDs in the entirety of Helsinki Capital Region, except in a square defined over the center of Helsinki, where the value is 2.5 minutes for all YKR IDs (Toivonen et al. 2014). See the switch "Helsinki walking center (TTM18)" in the toolbar to the left.
avg A mean duration (per postal code area, unit minutes) of the complete travel chain with private car from origin postal code area to the destination postal code area.
  • Formula: (car_x_t1 + car_x_t2 + ... + car_x_tn) / n, where x is r, m, or all. car_ represents unchanged TTM18 data columns.
drivetime The length of the driving segment (unit minutes) of the mean duration of the complete travel chain from origin postal code area to the destination postal code area without searching for parking (using the TTM18 value) or walking to the final destination of the travel chain (TTM18 value).
  • Formula: ttm_x_avg - ttm_sfp - ttm_wtd_avg, where x is r, m, or all.
pct How much does searching for parking and walking from one's parked car to the final destination of the travel chain constitute of the mean duration of the complete travel chain? Value per postal code area, unit percent, where 1 equals 100 %.
  • Formula: (ttm_sfp + ttm_wtd_avg) / ttm_x_avg, where x is r, m, or all.

Thesis data, row names

Row name Description
sfp_avg Time consumed, on average, in searching for parking in a postal code area, according to the thesis survey respondents. The thesis data variable parktime was averaged for the current postal code area for the use of this application.
  • Formula: (parktime1 + parktime2 + ... + parktimen) / n, while postal code value is constant.
wtd_avg Average walking time from one's parked private car to the final destination in a postal code area, according to the thesis survey respondents. The thesis data variable walktime was averaged for the current postal code area for the of this application.
  • Formula: (walktime1 + walktime2 + ... + walktimen) / n, while postal code value is constant.
drivetime The length of the driving segment (value per postal code area, unit minutes) of the mean duration of the complete travel chain (TTM18 data) from the origin postal code area to the destination postal code area, without searching for parking (thesis survey mean) or walking to destination (thesis survey mean). ttm_x_drivetime is used as the complete travel chain.
  • Formula: ttm_x_drivetime - thesis_x_sfp - thesis_x_wtd, where x is r, m, or all.
pct How much do the thesis survey mean values for searching for parking and walking from one's parked car to the final destination constitute of the mean duration of the complete travel chain (calculated from TTM18 data)? ttm_x_drivetime is used as the complete travel chain. Value per postal code area, unit percent, where 1 equals 100 %.
  • Formula: (thesis_x_sfp + thesis_x_wtd) / ttm_x_drivetime, where x is r, m, or all.

Comparison of TTM18 and thesis data, row names

Unit in comparison is percent, where 1 equals 100 %. Meanings of icons beside comparison values:
— Comparison value is less than 1, TTM18 data value is larger than thesis survey value
— Comparison value is more than 1, thesis survey data value is larger than TTM18 value
— Comparison value is 1, TTM18 and thesis survey data values are equal

Row name Description
sfp Compare TTM18 and thesis survey values for searching for parking. If this comparison value is smaller than one, searching for parking takes longer in TTM18. If value is larger than one, searching for parking takes longer in thesis data.
  • Formula: thesis_x_sfp / ttm_sfp, where x is r, m, or all.
wtd Compare TTM18 and thesis survey values for the duration to walk from one's parked car to the final destination of the travel chain.
  • Formula: thesis_x_wtd / ttm_wtd_avg, where x is r, m, or all.
drivetime Compare the driving time segment of the mean duration of the complete travel chain in TTM18 and thesis data.
  • Formula: thesis_x_drivetime / ttm_x_drivetime, where x is r, m, or all.
pct Compare the percentual value of the significance of searching for parking and walking to one's final destination of the mean duration of the complete travel chain in TTM18 and thesis data.
  • Formula: thesis_x_pct / ttm_x_pct, where x is r, m, or all.
