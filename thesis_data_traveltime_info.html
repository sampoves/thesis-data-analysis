Parking of private car in Helsinki Capital Region: compare Helsinki Region Travel Time Matrix 2018 and thesis survey results of Sampo Vesanen

Use these tables to learn about the abbreviations of the tooltip and the methodology how the values were calculated. All values presented were rounded to two decimal places.

Tooltip column names

Abbreviation Meaning
r Rush hour (09:00–11.00, 15.00–17.00)

TTM18, row names

Row name Contents
car_x_t The the entire travel time (per YKR ID, unit minutes) from origin YKR ID to destination YKR ID. Data directly from TTM18. Source columns are car_r_t, car_m_t, or car_sl_t for r, m, sl, respectively. NA indicates an non-navigable route.
sfp Time consumed in searching for parking. In TTM18 data, this is 0.42 minutes for all YKR IDs in the entirety of Helsinki Capital Region (Toivonen et al. 2014).
wtd_avg An averaged value (per postal code area, unit minutes) of walking time from one's parked private car to the final destination. This is 2.0 minutes for all YKR IDs in the entirety of Helsinki Capital Region, except in a square defined over the center of Helsinki, where the value is 2.5 minutes for an YKR ID (Toivonen et al. 2014).
avg An averaged value (per postal code area, unit minutes) of the complete travel time with private car from origin YKR ID to all destination YKR IDs in the current postal code area.
  • Formula: car_x_t1 + car_x_t2 + ... + car_x_tn / n, where x is r, m, or sl.
drivetime The length (value per postal code area, unit minutes) of the driving part of the complete travel time from origin YKR ID to the destination YKR ID with private car without searching for parking or walking to destination.
  • Formula: ttm18_x_avg - ttm18_sfp - ttm18_wtd_avg, where x is r, m, or sl.
pct How much does searching for parking and walking to one's destination constitute of the total drivetime? Value per postal code area, unit percent.
  • Formula: (ttm18_sfp + ttm18_wtd_avg) / ttm18_x_drivetime, where x is r, m, or sl.
