Parking of private car in Helsinki Capital Region: compare Helsinki Region Travel Time Matrix 2018 and thesis survey results of Sampo Vesanen

Focusing on private cars, this application visualises the parking time differences between Helsinki Region Travel Time Matrix 2018 (Tenkanen et al. 2018) and the survey data carried out for the master's thesis of Sampo Vesanen. Use this information dialog to learn how the tooltip values were calculated.

Tooltip column names

Column name Description
r Rush hour traffic (09:00–11.00, 15.00–17.00)
m Midday traffic (09.00–15.00)
sl Route following speed limits without any additional impedances. NB! For thesis data, this consists of timeofday: weekend data.

TTM18, row names

Row name Description
car_x_t Travel time from origin to destination by private car, the whole travel chain has been taken into account. This is unchanged TTM18 data. Replace x in the row name with r, m, sl to receive car_r_t, car_m_t, or car_sl_t, respectively. NA indicates a non-navigable route.
sfp Time consumed in searching for parking. In TTM18, this is 0.42 minutes for all YKR IDs in the entirety of Helsinki Capital Region (Toivonen et al. 2014).
wtd_avg An averaged value (per postal code area, unit minutes) of walking time from one's parked private car to the final destination. In TTM18 this is 2.0 minutes for all YKR IDs in the entirety of Helsinki Capital Region, except in a square defined over the center of Helsinki, where the value is 2.5 minutes for an YKR ID (Toivonen et al. 2014).
avg An averaged value (per postal code area, unit minutes) of the complete travel time with private car from origin YKR ID to all destination YKR IDs in the current postal code area.
  • Formula: car_x_t1 + car_x_t2 + ... + car_x_tn / n, where x is r, m, or sl.
drivetime The length of the driving part (value per postal code area, unit minutes) of the averaged complete travel time from origin YKR ID to the destination YKR IDs without searching for parking (TTM18 value) or walking to destination (TTM18 value).
  • Formula: ttm18_x_avg - ttm18_sfp - ttm18_wtd_avg, where x is r, m, or sl.
pct How much does searching for parking and walking from one's parked car to the final destination constitute of the total drivetime? Value per postal code area, unit percent.
  • Formula: (ttm18_sfp + ttm18_wtd_avg) / ttm18_x_drivetime, where x is r, m, or sl.

Thesis data, row names

Row name Description
sfp_avg Time consumed, on average, in searching for parking in different areas of Helsinki Capital Region, according to the thesis survey respondents. The values of thesis data variable parktime are averaged for the current postal code area.
  • Formula: parktime1 + parktime2 + ... + parktimen / n, while postal code value is constant.
wtd_avg Average walking time from one's parked private car to the final destination in different areas of Helsinki Capital Region, according to the thesis survey respondents. The values of thesis data variable walktimeare averaged for the current postal code area.
  • Formula: walktime1 + walktime2 + ... + walktimen / n, while postal code value is constant.
drivetime The length of the driving part (value per postal code area, unit minutes) of the mean travel time from origin YKR ID to the destination YKR IDs, without searching for parking (thesis survey mean) or walking to destination (thesis survey mean).
  • Formula: ttm_x_avg - thesis_x_sfp - thesis_x_wtd, where x is r, m, or sl.
pct How much do the thesis mean values for searching for parking and walking from one's parked car to the final destination constitute of the total drivetime, calculated from TTM18 data? Value per postal code area, unit percent.
  • Formula: (thesis_x_sfp + thesis_x_wtd) / ttm_x_avg, where x is r, m, or sl.

Comparison of TTM18 and thesis data, row names

Meanings of icons beside comparison values:
— Comparison value is less than 1, TTM18 data value is larger than thesis survey value
— Comparison value is more than 1, thesis survey data value is larger than TTM18 value
— Comparison value is 1, TTM18 and thesis survey data values are equal

Row name Description
sfp Compare TTM18 and thesis survey value for searching for parking. If this comparison value is smaller than one, searching for parking takes longer in TTM18. If value is larger than one, searching for parking is longer in thesis data.
  • Formula: thesis_x_sfp / ttm_sfp, where x is r, m, or sl.
wtd Compare TTM18 and thesis survey value for the duration to walk from one's parked car to the final destination of the journey.
  • Formula: thesis_x_wtd / ttm_wtd_avg, where x is r, m, or sl.
drivetime Compare the driving time part of the travel chain in TTM18 and thesis data.
  • Formula: thesis_x_drivetime / ttm_x_drivetime, where x is r, m, or sl.
pct Compare the percentual value of significance of searching for parking and walking to one's destination in TTM18 and thesis data.
  • Formula: thesis_x_pct / ttm_x_pct, where x is r, m, or sl.
