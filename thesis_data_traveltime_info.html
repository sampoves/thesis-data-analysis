Tooltip TTM18 row names

Row name Contents
sfp Time consumed in searching for parking. Averaged per postal code area.
wtd_avg Walking time to the final destination. Averaged per postal code area.
avg The complete travel time from origin YKR ID to the destination YKR ID, sfp and wtd included, averaged for the current postal code area. Value in minutes.
drivetime The driving part of the complete travel time from origin YKR ID to the destination YKR ID. sfp and wtd are substracted from the total travel time. Value in minutes.
pct How much does sfp and wtd constitute of the total travel time. A percentage value.
